## Commit Hash Note

A Git commit cannot contain its own final SHA without a follow-up amend cycle,
because the SHA is calculated from the committed tree. The JSON report therefore
uses `final_commit_reported_by_codex` as an explicit self-reference note. The
actual final commit SHA is reported in the final Codex response after commit and
push.
