# Frontend API v2 SP37 Secret Scan

Raw staged-diff scan pattern:

```text
(password|secret|token|authorization|set-cookie|cookie:)
```

Result:

```text
PASS_WITH_FALSE_POSITIVES
```

Classification:

- `KORNIX_FRONTEND_SMOKE_PASSWORD` is an environment variable name documented
  for the live smoke runner; no password value is committed.
- `csrfToken`, `X-CSRF-Token` and `set-cookie` are source-code identifiers used
  by the smoke runner and HTTP client; no CSRF token or cookie value is
  committed.
- Report text states that cookies, CSRF tokens and passwords are redacted; it
  does not contain their values.

No literal credential, authorization header value, cookie value, CSRF token
value, `.env` secret or backend dump is staged.
